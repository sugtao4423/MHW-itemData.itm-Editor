# MHW itemData.itm Editor
But Android Application

You can download from release page

Don't need original `itemData.itm` file

## Built-In itemData.itm
* chunk0
* chunk3
* chunk5
* chunk7
* chunk9

## features
* view items
* search items by item name
* edit carry, sell and buy values
* save

## Languages
* English (default)
* Japanese
